# 중앙계좌 위험·자금배분 기준선

상태: `RESEARCH_ONLY / LIVE_RISK_0 / BANK_OFFLINE / GLOBAL_SLOT_1`

## 동결 계약

- 운용 종목: BTCUSDT, ETHUSDT, SOLUSDT, XRPUSDT
- 전역 슬롯: 미체결 진입 주문 + 보유 포지션 합계 최대 1
- 총자산: 거래소 자동매매계좌 + 운영자가 수동 기록한 은행계좌
- 은행 연결: 없음
- 은행 이체: 프로그램은 지시만 생성하며 실제 이체는 운영자가 수동 수행
- 거래소 후보: Binance USD-M 또는 Bybit linear
- 수량: 사용자가 지정한 고정손실 공식으로 계산 후 exchange quantity step에 내림
- 용량 부족: 수량 축소 금지, 주문 차단 및 필요한 수동 입금액 출력

## 현재 소표본 후보의 위험 진단

입력은 ETH midnight PDH/PDL raid 후보의 두 청산 arm이다. 두 원장 모두 2020-01-11부터 2025-11-26까지 42거래이며, 완전 운용일 2,147일 대비 거래 빈도는 0.01956회/일이다. 권장 1회/일에 크게 미달하므로 주전 데이트레이딩 전략으로 승격할 수 없다.

### 1.25R arm

- 연구상 위험률 후보: 1.25%
- 관측 최종 총계좌배수: 1.21750
- 관측 최대낙폭: 4.90%
- 20,000회 분기 block bootstrap 중앙 최종배수: 1.19920
- bootstrap p05 최종배수: 1.06708
- bootstrap p95 최대낙폭: 5.87%
- 거래소 목표 비율: 61.87%
- 은행 목표 비율: 38.13%

### 1.50R arm

- 연구상 위험률 후보: 1.25%
- 관측 최종 총계좌배수: 1.26570
- 관측 최대낙폭: 4.90%
- 20,000회 분기 block bootstrap 중앙 최종배수: 1.24154
- bootstrap p05 최종배수: 1.08331
- bootstrap p95 최대낙폭: 5.99%
- 거래소 목표 비율: 61.87%
- 은행 목표 비율: 38.13%

## 해석

현재 1.25%는 전략 승격값이 아니라 5배 레버리지 상한, 25% 증거금 buffer, 최대손실 2배의 거래소 buffer, 최소 은행보유 35%를 함께 적용했을 때 이 소표본 원장에서 허용되는 연구상 용량 최적값이다. 1.50% 이상은 p99 주문 geometry에서 최소 은행보유 35%를 깨므로 제외된다.

전략의 거래 수와 빈도, 독립 OOS, 체결·shadow 증거가 부족하므로 실제 배포 위험률은 0%다. 향후 동결 전략이 별도 승격 gate를 통과한 경우에만 같은 optimizer를 새 원장에 다시 적용한다.
