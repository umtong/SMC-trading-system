# SMC/ICT 2호기 — 평가 계약

## 목표

- 계좌 NAV 기준 비용 후 일평균 기하 복리성장률 1% 이상
- 강제청산과 회복 불가능한 계좌 훼손 회피
- 낮고 회복 가능한 최대낙폭
- 다수의 실질적으로 독립적인 거래에서 반복되는 양의 기대수익
- 현실적 체결과 변화한 시장 레짐에서도 유지되는 실사용 가능성

## 즉시 무효 조건

- 미래정보, 룩어헤드 또는 당시 이용 불가능한 정보
- 사후 종목·기간 선택이나 동일 구간 재사용에 따른 누수
- 계산·데이터 오류
- 실제로 획득할 수 없는 체결가격
- 중요 비용·펀딩·슬리피지·지연의 누락
- 결과 확인 후 규칙·평가 구간·판정 기준의 유리한 변경
- 탐색량을 숨긴 선택적 결과 보고

## 기본 보고 지표

- 비용 후 기하 성장률과 계좌 배수
- 최대낙폭과 회복기간
- 청산 가능성, 손실 꼬리와 연속 손실
- 거래 수, 거래당 기대값과 빈도
- 상위 1%·5%·10% 거래 기여도와 제거 후 성과
- 종목·기간·방향·레짐별 기여도
- 수수료·스프레드·슬리피지·지연·부분체결·시장충격 민감도
- 파라미터와 데이터 구간 변화에 대한 안정성
- 독립 전략 간 손익 상관과 동시 손실

전략 특성상 불필요한 지표는 생략할 수 있지만, 생략으로 핵심 취약점이 숨겨져서는 안 된다.

## Champion 비교

서로 다른 데이터·비용·체결·평가 기준의 결과를 직접 순위 비교하지 않는다. 비교 가능한 조건으로 정규화하거나 차이를 명시한다. 새 후보가 현 Champion보다 목표 달성 가능성을 실질적으로 높인다는 근거가 있을 때만 교체한다. 교체 전에 기존 Champion의 코드·데이터·파라미터·비용·결과 위치를 보존한다.

## 검증의 위치

기본 유효성은 모든 전략이 처음부터 충족해야 한다. 오류는 신속히 수정·무효화하고 실제 비용 후 알파 탐색으로 복귀한다. 검증은 수익성 연구를 보호하는 조건이며 프로젝트의 대리목표가 아니다.
